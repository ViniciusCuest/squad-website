MetaTech EDU

Projeto baseado em Realidade virtual MetaTech EDU

O projeto é composto composto por uma aplicação web onde tem todo os recursos disponíveis para capacitação profissional da educação através de Realidade Virtual. Utilizamos a ferramenta Figma para fazer a prototipação da nossa plataforma. O Figma é uma ferramenta gratuita que permite o compartilhamento dos projetos, o uso colaborativo para desenvolvimento de prototipagens e interfaces gráficas para desenvolvimento de experiência de usuário. Foi escolhida esta ferramenta por ser gratuita e facilitar a colaboração de desenvolvimento dos protótipos, uma vez que todo o trabalho é armazenado em nuvem e podendo ser desenvolvido em conjunto pela equipe.

Manual Simplificado - Uso da marca

Disponibilização do manual simplicado da marca, onde se encontra as paletas de cores, bem como seus código hexadecimais, utilizadas na criação logotipo da marca da equipe e do corrente projeto (MetaTech EDU), além dos diferentes cenários em que podem ser utilizadas como no modo light ou dark assim proporcionando uma melhor expêriencia de usuário (UX).
